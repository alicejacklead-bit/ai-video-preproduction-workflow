# Workflow Blueprint

Use this sequence as the default production order:

1. Creative brief
Capture format, platform, duration, ratio, target audience, hook, conflict, payoff, and tone.

2. Plan Demo
Write the director-facing plan in prose before generating assets.

Required fields:
- title or working name
- duration
- aspect ratio
- platform
- genre or content type
- core beat structure
- character count
- main location or world
- visual goal
- ending payoff

3. Character board
Lock identity features that must survive generation:
- face shape
- hairstyle
- body type
- clothing
- accessories
- signature expressions
- signature gestures

4. Scene layout
Lock geography:
- entrances and exits
- hero object positions
- furniture or environment anchors
- movement paths
- light direction
- background style constraints

5. Camera blocking
Assign camera responsibilities before storyboarding:
- establishing camera
- reverse-shot cameras
- special reveal or insert camera
- movement restrictions
- screen direction rules

6. Storyboard
Map each beat to one camera and one narrative job. Avoid ambiguous shots that ignore the blocking map.

7. Package board
Assemble the most useful control frames and labels into one handoff board for the model.

8. Final prompt
Translate the locked decisions into generation language. Repeat fragile continuity constraints directly in the prompt.

## Minimum Deliverable

If the user asks for a lightweight workflow, still output:
- brief
- Plan Demo
- continuity constraints
- final prompt

If the user asks for a full workflow, output all eight steps above.
