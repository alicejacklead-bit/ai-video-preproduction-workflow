# 封装图提示词

## 用途

把这条 15 秒视频需要的核心控制信息装进一张图里，方便统一喂给模型，也方便你后续做教程拆解。

## 封装图提示词

高质量 3D 动画化日漫科幻动作导演封装图，主题是一名银白色高马尾、紫色发光眼睛的女性精灵快递员，在阴暗潮湿的老地铁隧道里驾驶深蓝色未来摩托，被黑衣追兵高速追逐，并在迎面列车到来前沿列车外侧狭缝像素化贴边穿越。角色与镜头风格接近高规格动画电影追逐段落，而不是二维番剧或真人电影海报。  
整张封装图采用清晰的导演提案板布局，而不是海报。画面分为四个信息区块：

1. 左上：人物图区  
展示女主与追兵的小型角色设定图，强调女主银白高马尾、紫色发光眼睛、黑银夹克、深蓝摩托；追兵秃头、黑色半脸呼吸面罩、黑灰骑行服、哑光黑重型摩托。

2. 右上：场景图区  
展示同一条老旧地铁隧道，潮湿地面反光、顶部连续灯带、轻雾、少量真实涂鸦和旧墙污渍，冷青绿色环境光与暖黄色列车前灯形成对比，空间狭长压迫，背景有动画电影式景深和运动柔化。

3. 左下：机位图区  
俯视导演机位图，清楚标出女主、追兵、迎面列车、狭缝通过路线、安全闸门位置，用箭头标出追逐方向和列车方向，明确女主是沿列车外侧与隧道墙之间的狭窄物理缝隙通过，不是穿过列车实体。

4. 右下：核心分镜图区  
四格时间轴分镜：
- 开场低机位追逐
- 后视镜追兵逼近与前方列车出现
- 女主像素化贴边穿越列车侧缝
- 追兵急刹，女主回头按下按钮，安全闸门落下

整体风格：黑灰底、青蓝色技术标记、暖黄色危险高光、信息层级清楚、像导演提案板，不是企业 PPT，也不是纯视觉海报。封装图中的示意画面要体现高质量 3D 动画角色、强透视、近景角色清晰和背景速度虚化。  
要求标签可读、箭头清晰、布局整洁、有速度感但不杂乱。

## 负面限制

- 不要做成单纯电影海报
- 不要省略机位图区
- 不要把像素化通过画成瞬移
- 不要加第三名骑手
- 不要随机出现站台或地面街道
- 不要过度花哨到信息看不清
