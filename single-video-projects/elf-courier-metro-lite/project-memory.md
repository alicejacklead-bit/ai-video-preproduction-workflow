# 已确认决定

- 输出模式：轻量模式
- 项目属性：单条视频项目
- 平台：抖音
- 时长：15 秒
- 画幅：4:3
- 叙事方式：同一条地铁隧道内的一次连续追逐
- 核心反转：女主不是穿过列车实体，也不是瞬间传送，而是在撞上前短暂像素化，沿着列车外侧与隧道墙之间的狭窄物理缝隙贴边通过
- 结尾动作：女主通过后按下按钮，安全闸门在追兵面前垂直落下

# 已锁定角色

- 女主：年轻女性精灵，银白色长发高马尾，细长精灵耳，发光紫色眼睛，黑银色短款未来机车夹克，青蓝色发光纹路，黑色短裙式战术下装，深色长靴
- 追兵：成年男性，剃光头，黑色半脸呼吸面罩，深灰黑色骑行服

# 已锁定场景

- 全片只在一条老旧地铁隧道内发生
- 墙面潮湿、脏旧、有真实霓虹涂鸦
- 空气里有灰尘、雾气、湿冷感
- 不切换到站台、地面街道、室外或其他空间

# 已锁定关键载具与道具

- 女主摩托：深蓝色流线型未来运动摩托，带青蓝色发光线条
- 追兵摩托：哑光黑色重型运动摩托
- 地铁列车：老旧金属车身，暖黄色强前灯，迎面高速驶来
- 安全闸门：工业感强的地铁隧道垂直落闸

# 已批准机位逻辑

- 全片保持同一条追逐轴线
- 先交代追逐空间，再交代列车威胁，再交代像素化通过，再交代追兵被拦截
- 追兵始终在女主身后，直到最后被截断
- 列车始终从女主前方迎面驶来

# 已批准故事结构

- 开场：高速追逐
- 中段：迎面列车出现，危险升级
- 反转：女主像素化贴边穿越
- 收尾：追兵急刹，女主落闸截断

# 已批准输出模式

- 轻量模式

# 用户修正

- 不要加油站
- 不要火箭
- 不要科技手雷
- 不要第三名骑手
- 不要改变发型、服装和摩托颜色
- 不要让人物弃车
- 不要出现游戏界面、血腥、字幕、无意义魔法光效
- 不要让人物真正穿过列车实体内部
- 不要出现无法理解的瞬间传送
- 不要过度切镜

# 可复用规则

- 只要逃脱机制非常规，就必须在方案和提示词里重复说明它的物理路径
- 追逐短片必须先交代空间，再上特效，不然观众会看不懂

# 已否决方向

- 炫技式乱切
- 无逻辑魔法化
- 随机切换隧道空间
